PM Orchestration API

Product Management orchestration tools and document generation APIs.

Available Endpoints

POST /api/ai-generate

AI-powered content generation endpoint.

Request Example:

{
  "system": "You are a helpful product management assistant.",
  "user": "Generate a PRD outline for a new feature."
}

Response Example:

{
  "output": "# Product Requirements Document\n\n## Overview\n\n..."
}

Set OPENAI_API_KEY environment variable for the API to work.
